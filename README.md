# ESP32 Weather Station

## Overview
A virtual weather station built using ESP32 and DHT22 in the Wokwi simulator.

## Features
- Measures temperature
- Measures humidity
- Displays sensor readings on the Serial Monitor

## Hardware
- ESP32 DevKit V1
- DHT22 Sensor

## Software
- Arduino IDE
- Wokwi Simulator

## Current Status
Project completed and tested successfully in simulation.
